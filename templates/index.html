About the Bot

Icarix Translator Bot automatically translates messages between users who speak different languages in Telegram group chats.

Features

  • Automatic language detection
  • Real-time message translation
  • Topic-specific activation
  • Support for 100+ languages
  • Admin-controlled activation

How to Use

  1. Add to Group: Add @Icarix_Translator_Bot to your Telegram group
  2. Activate: An admin types /start in the desired topic
  3. Start Chatting: The bot detects languages and translates automatically

Commands

Command Description
/start Activate the bot in a topic (admin only)
/help Display help information
/language [code] Manually set your language
